# DancerGirl Content Hierarchy & Mapping

## Simple Structure - How Sanity Maps to Frontend

### 📰 STORIES (Main Content)
**Sanity Location:** Stories → All Stories  
**Frontend:** 
- Homepage: "Latest from DancerGirl" section
- Stories page: All stories listed here
- Individual story pages

**Content Type:** `article`

#### Sub-sections:
1. **⭐ Featured Stories**
   - Stories with "Featured" checkbox ON
   - Shows first in "Latest from DancerGirl" on homepage
   
2. **🔥 Trending Stories**  
   - Stories with "Trending" checkbox ON
   - Can add trending badge on frontend

3. **📚 By Category**
   - Organize stories 
   
    topics (e.g., "Money Moves", "Dancer Speak Up")

### To Create a New Story:
1. Go to **Stories → All Stories**
2. Click "+ Create"
3. Fill in:
   - Title
   - Excerpt (preview text)
   - Main Image
   - Body content
   - **Check "Featured" to show in Latest from DancerGirl**
   - Select Categories

### To Move Renee's Story:
1. Create new story in **Stories → All Stories**
2. Copy content from old Featured Story
3. Check the "Featured" box
4. Delete old featuredStory document
